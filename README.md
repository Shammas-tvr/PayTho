# PayTho ERP Backend

PayTho is a SaaS-based ERP system designed to manage multi-company business operations including branches, users, billing, stock, and reporting.

## Tech Stack
- Python
- Django
- PostgreSQL (planned)
- REST APIs (upcoming)

## Current Features (Phase 1 – Foundation)
- Custom User Model with role-based access
- Multi-company architecture
- Branch management
- Django admin configuration

## Upcoming Features
- Separate developer and company login
- Role-based dashboards
- Product & billing modules
- Reporting system

## Project Status
🚧 Under active development (Phase 1)
